Oracle Web Token

Introducing a new secure method of sharing sensitive data between two parties!

Oracle Web Token (OWT) uses builtin in military grade encryption for sharing sensitive data that the recipient cannot read.

To start using OWT, simply download the OWT python module from here, install pycryptodome and import it in your projects!

To verify our claim that OWT is secure, this demo API returns a OWT token with a hidden flag in the following format. If you can find a way to view the flag we will give you a big prize!

                    
{
    "flag" : "FAKE{the_API_stores_the_actual_flag}"
}

GET /api/token/create

Returns a OWT token with the hidden flag.

Example Response

                    
{
    "status":"success",
    "value":{
        "token":"eyJpdiI6ICJBRnloZVVKc05hdHY3U1FPTzMwN3hBIn0.4ISVfG3FcOOAi-0aTn7tv7UFIkdV0WLcyAGoLoEC5p6oYbO7WJpNfV53GJii_v6yAeUWlQkv0TDzg2Wd0cx6kQ"
    }
}

POST /api/token/verify

Verifies the OWT token is a valid. If it is not valid then the API will inform you of the reason why.

Example JSON POST Body

                    
{
    "token":"eyJpdiI6ICJBRnloZVVKc05hdHY3U1FPTzMwN3hBIn0.4ISVfG3FcOOAi-0aTn7tv7UFIkdV0WLcyAGoLoEC5p6oYbO7WJpNfV53GJii_v6yAeUWlQkv0TDzg2Wd0cx6kQ"
}

Example Response

                    
{
    "status":"success",
    "value":"Token is a valid OWT token!"
}
